What Are Views and Stored Procedures?
View = A saved query that acts like a virtual table

You query it like a normal table
Always shows fresh data
Simplifies complex queries

Stored Procedure = A saved SQL program with parameters

You call it with different inputs
Like a function in Python
Used heavily in real banking systems
